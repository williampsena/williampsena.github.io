Atualizando seu Linux Debian

by William Sena

2015-10-04

O que é?

É um sistema operacional Open Source que atualmente utiliza o Kernel do Linux ou FreeBSD Kernel.

Existem muitas distribuições voltadas para Desktop que são baseadas no Debian, devido sua estabilidade e atualizações. O Ubuntu é uma das distribuições mais famosas e foi baseada no Debian, seguindo a lista Mint, ElementaryOS, Xubuntu, Kubunt e Lubuntu, também são versões baseadas no Ubuntu, ou seja, temos muitas versões baseadas no Debian Style.

É seguro atualizar?

Para garantir a segurança em atualizações é recomendável utilizar repositórios oficiais de pacote. Sempre que for necessário utilizar um repositório não oficial, verifique a compatibilidade com a sua versão.

Qual é versão do meu Debian?

Se você seguiu os meus caminhos e deixou seu linux desatualizado por volta de um ano, você deve estar com a versão Debian Whezzy, agora se você estiver com a versão Squeezy ou Lenny, que feio vamos atualizar isso.

Entre em sua linha de comando e digite o comando abaixo para identificar a versão do seu linux.

lsb_release -a

O processo de atualização é seguro, porém recomendo que você faça um backup dos seus arquivos pessoais.

Versões do Debian.

O Debian mantém seu trabalho distribuído em três versões.

  • Stable - Última versão lançada oficialmente.
  • Testing - Versão que contém os pacotes que ainda não foram aceitos na versão oficial, porém é uma versão mais atualizada.
  • Unstable - Versão com pacotes que ainda estão em desenvolvimento. Exclusivo para desenvolvedores que querem uma nova experiência com emoções.

Em 2015 a versão Jessie foi oficialmente lançada.

  • Debian 8 Jessie (stable)
  • Debian 9 Stretch (testing)
  • Debian 10 Sid (unstable)

Últimas versões oficiais obsoletas

  • Debian 7 Wheezy
  • Debian 6 Squeeze
  • Debian 5 Lenny

Mantendo a versão atual atualizada

Será necessário executar um procedimento recomendado para garantir que todos os pacotes estão na última versão, antes de realizar um "upgrade" para a próxima versão, digite os comandos abaixo:

Somente o usuário root tem este privilégio. Utilize o sudo para execução ou efetue o login como root

apt-get update
apt-get upgrade  
apt-get dist-upgrade

Mudando os repositórios

O gerenciamento de pacotes é realizado pelo apt, o arquivo de registro dos pacotes está localizado em /etc/apt/source.list.

Será necessário alterar a versão do repositório, siga a tabela abaixo que indica a próxima versão disponível para atualização.

Atual Próxima
Squeeze Wheezy (Stable)
Wheezy Jessie (Stable)
Jessie Stretch (Testing)

Você pode alterar o arquivo de forma simples digitando o comando abaixo.

O comando abaixo altera a versão de wheezy para jessie

sed -i 's/wheezy/jessie/g' /etc/apt/sources.list

Você também pode alterar manualmente utilizando o nano, digite em sua linha de comando.

sudo nano /etc/apt/source.list

Arquivo de exemplo:

deb http://ftp.us.debian.org/debian/ stretch main non-free
deb-src http://ftp.us.debian.org/debian/ stretch main non-free
deb http://security.debian.org/ stretch/updates main non-free
deb-src http://security.debian.org/ stretch/updates main non-free
deb http://ftp.us.debian.org/debian/ stretch-updates main non-free
deb-src http://ftp.us.debian.org/debian/ stretch-updates main non-free

deb http://ftp.de.debian.org/debian stretch main contrib non-free
deb-src http://ftp.de.debian.org/debian stretch main contrib non-free

Para salvar pressione ctrl+x.

Atualizando sua distribuição

Agora que os repositórios foram atualizados vamos ao upgrade!

apt-get update
apt-get upgrade
apt-get dist-upgrade

Este processo envolve download de pacotes, instalação, compilação, alterações de script, atualização de Kernel, ou seja, recomendo que você execute quando tiver um tempo, pois é processo demorado.

Após a atualização, reinicie o sistema digitando:

sudo reboot

Raspberry

A distribuição Debian do Raspberry é o Raspbian, uma versão baseada no Debian Wheezy.

Você pode realizar o mesmo procedimento para atualizar da versão Wheezy para a versão Jessie, claro o repositório do Raspberry é diferente.
